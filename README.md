# Weather App

A simple Spring Boot application that fetches and displays hourly temperature forecast for Toronto for the next 7 days.

---

## How to Run

1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Navigate into the project directory:
   ```bash
   cd <your-project-folder>
   ```
3. Build and run the app:
   ```bash
   ./mvnw spring-boot:run
   ```
   (or `mvn spring-boot:run` if you have Maven installed)

4. Open your browser and go to:
   ```
   http://localhost:8000
   ```

---

## What it Shows

- Hourly temperature forecast for the next 7 days.
- Data fetched from the Open-Meteo API.
- Displays temperature for each hour during the forecast period.
